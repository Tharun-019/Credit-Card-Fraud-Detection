========================================================
FRAUDX
AI-POWERED CREDIT CARD FRAUD DETECTION SYSTEM
========================================================

Project Name:
FraudX

Project Type:
Machine Learning based Web Application for Fraud Detection

Developed Using:
Flask, Python, SQLite, SQLAlchemy, HTML, CSS, JavaScript,
Scikit-learn, Pandas, NumPy, Joblib, Matplotlib

========================================================
1. PROJECT OVERVIEW
========================================================

FraudX is an AI-based credit card fraud detection system developed
to identify fraudulent transactions in real-time. The system allows
users and administrators to upload datasets, train machine learning
models, and detect fraud based on transaction details.

The system provides the following modules:

1. User Login System
2. Dataset Upload
3. Model Training
4. Manual Prediction
5. Fraud Detection Result
6. Analytics and Charts
7. Performance Analysis

========================================================
2. MAIN OBJECTIVE
========================================================

The main objective of FraudX is to detect fraudulent credit card
transactions using machine learning techniques and provide accurate
results for secure financial systems.

7. HOW TO RUN THE PROJECT
========================================================

STEP 1:
Open the project folder.

STEP 2:
Ensure Python is installed.

STEP 3:
Install required libraries:

pip install flask flask-sqlalchemy pandas numpy scikit-learn joblib matplotlib

STEP 4:
Run the application:

python app.py

STEP 5:
Open browser and go to:

http://127.0.0.1:5000/

========================================================
8. HOW TO USE THE PROJECT
========================================================

STEP 1: Open Home Page
- Launch the project in browser.

STEP 2: Login
- Enter username and password.

STEP 3: Upload Dataset
- Upload CSV file containing transaction data.

STEP 4: Train Model
- Click "Train Model" button.
- System trains model using dataset.

STEP 5: Manual Prediction
- Enter transaction details manually.

STEP 6: Submit Input
- Click predict button.

STEP 7: View Result
- System displays whether transaction is:
  Fraudulent or Legitimate.

STEP 8: View Analytics
- Check charts and performance results.

STEP 9: Logout
- Exit the system safely.
